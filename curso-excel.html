Curso Profesional de Microsoft Excel

Gestión, Automatización y Análisis de Datos para Negocios

Optimiza el control administrativo de tus proyectos. Este curso está diseñado desde una perspectiva 100% práctica, utilizando problemas reales de negocios para dominar la herramienta analítica más potente del mundo profesional.

Plan de Estudio Detallado

Unidad 1: Fundamentos y Estructura Esencial

  • Interfaz de trabajo, celdas, filas, columnas y formatos de datos (Texto, Número, Moneda, Fecha).
  • Operaciones aritméticas básicas y uso de referencias absolutas ($A$1) para fijar datos.
  • Diseño de tablas organizadas y uso de filtros/ordenamiento rápido.

Unidad 2: Fórmulas y Funciones Automatizadas

  • Uso de funciones estadísticas y lógicas básicas (SUMA, PROMEDIO, CONTAR, SI).
  • Funciones de búsqueda indispensables (BUSCARV, BUSCARX) para enlazar inventarios o bases de datos.
  • Gestión de texto y fechas para control de plazos o clientes.

Unidad 3: Análisis Visual y Reportes

  • Creación e interpretación de Gráficos Estadísticos (Barras, Líneas, Pasteles).
  • Introducción a Tablas Dinámicas para resumir grandes volúmenes de ventas o gastos en segundos.
  • Diseño de reportes limpios y listos para impresión o exportación a PDF.
